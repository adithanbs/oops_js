1. Object– An Object is a unique entity that contains property and methods. 
For example “car” is a real life Object, which has some characteristics like color, 
type, model, horsepower and performs certain action like drive. 
The characteristics of an Object are called as Property, in Object-Oriented Programming and 
the actions are called methods. An Object is an instance of a class. Objects are everywhere in JavaScript
almost every element is an Object whether it is a function, array, and string.
